# **App Name**: Menu Magic

## Core Features:

- Menu Display: Display menu items with name, description, and price.
- Menu Sections: Categorize menu items into relevant sections (e.g., Espresso Bar, Craft Coffees).
- Search and Filter: Allow users to quickly find menu items.
- AI-Powered Recommendations: Suggest menu items based on user preferences and popular choices; uses a tool to reason whether or not to return a specific item.

## Style Guidelines:

- Primary color: Sienna (#A0522D) for a warm, inviting feel.
- Background color: Beige (#F5F5DC), very desaturated to create a subtle backdrop.
- Accent color: BurlyWood (#DEB887) as a lighter analogous color for highlights.
- Body font: 'Montserrat' sans-serif for a clean, modern look in item descriptions and prices.
- Headline font: 'Great Vibes' script for headings to create a decorative, inviting presentation.
- Use minimalist, linear icons to represent categories (coffee, food, etc.).
- Follow a clear, well-structured layout, similar to the reference design. Divide items into sections with visual separators. Prioritize readability and easy navigation.
